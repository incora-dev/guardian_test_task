## Config

Create .env file in root of project as .env.example and add missed values

# Run to install all dependencies
```bash
$ yarn install 
``` 

# Commands to run app 
```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
# Commands to run tests

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
